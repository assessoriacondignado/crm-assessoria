API M2M — Endpoints Disponíveis

Integre qualquer plataforma de IA ou automação usando nossa API REST simples com autenticação Bearer Token.

POST consulta_completa
Fluxo Completo — Consentimento + Margem + Simulação

Ação principal. Envia o consentimento para a Dataprev, aguarda a margem e já retorna a simulação padrão. Se a Dataprev demorar, retorna AGUARDANDO_DATAPREV com instrução de retry.

CPFTelefoneBearer Token
POST simular_proposta
Simulação Personalizada

Refaz a simulação com parâmetros customizados: prazo específico, valor de parcela desejado ou valor líquido. Ideal para o robô apresentar opções ao cliente.

PrazoValor ParcelaValor Líquido
POST enviar_proposta
Gerar Link de Assinatura Digital

Registra a proposta na V8 e retorna o link de formalização. O cliente acessa o link e assina digitalmente sem precisar ir a uma agência.

PIXLink AssinaturaNúmero Proposta
