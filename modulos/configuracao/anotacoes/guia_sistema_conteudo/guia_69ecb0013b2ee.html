1. Ambiente feito para o usuário criar e configurar campanhas de vendas
  • Acesse o menu lateral para entrar no ambinte Campanhas
  • O usuário precisa ter permissão para o acesso
  • O ambiente possui restrições de hierarquia

2. Acessando a campanha
  • Para o usuário começar a trabalhar na campanha, ele deve:
    •  Clicar em inicio
    • Escolher a campanha
    • O CRM vai direcionar para o MODO CAMPANHA
