1. REGRAS
  • Essa integração necessita que o usuário possua salvo para consulta
  • A utlização da integração é registrada vi Financeiro
2. FLUXO
  • Ao acessar o cleinte o usuário deve clicar na integração CONSULTA INSS (HIST)
  • O CRM vai atualizar a ficha cadastras
    • Dados Cadatrais
    • Dados do benefício

3. INFORMAÇÕES DO BENÉFICIO
  • O sistema vai apresentar a data que foi feito a consulta na integração e a data que o NB teve a sua utlima versão atualizada
  • Calculo de margem: o sistema vai apontar a informação sobre a regra de Taxa de juros
    • Se o banco Registrar o valor liquido no historico, o sistema calcula a taxa de juros padrão apra o período
  • Data consulta: é a data que foi feito a consulta na integração
  • Data atualização NB: é a data que o NB foi atualizado no banco de dados
