1. BOTÃO DE CAMPANHA PARA O CPF
  • Permite o usuário aplicar comando aos CPF
  • Permite o usuário Verificar registros para  CPF do cliente
  • O CRM possui regras de restrição, que podem impedir do usuário ver o registro feito por outros usuários na mesma empresa

2. DADOS CADASTRAIS
  • Dados como CPF, Sexo, mãe, pai e outros dados

3. DADOS DE  TELEFONE, ENDEREÇO E-MAIL
  • Botão ligar, permite o usuário direcionar a ligação para seu celular se o mesmo estiver conectado ao PC

4. DADOS DE CONVENIOS E CONTRATOS
  • Se trata do historico do cliente e as inforamações do benefício
  • Cosnulte o suporte para saber qual convênio está liberado para seu usuário

5. BOTÕES DE ACESSO A INTEGRAÇÕES
  • Permite fazer a atualização do cliente por outros sistema externo
  • Seu usuário precisa está habilitado


6. BOTÃO HISTÓRIO DE CONSULTA
  • Permite o usuário acessar todo o CPF consultado pelo seu usuário

7. BOTÃO DADOS DE ORIGEM
  • Utilizado para pesquisa, onde fica o registro das importações e criação de agrupamento pelo usuário


8. RODA PÉ
  • Registra todas as informações de cosnulta feita pelo seu usuário
  • Depende do nível de permite, pode resringir o usuário para consultar somente o seus registros
