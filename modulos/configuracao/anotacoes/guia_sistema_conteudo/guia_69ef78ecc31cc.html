2. LISTA DE CAMPANHAS
  • Ao acessar as campanhas, o CRM vai apresentar a lista de campanahs disponíveis
    • Nome da Campanha:
      • O usuário pode criar o nome de sua campanha
    • Empresa Vinculada: Nome da empresa
      • É o nome da empresa que o usuário faz parte
    • Período (Início - Fim)
      • Periodo que a Campanha fica disponível para o usuário acessar
    • Usuário(s) Vinculado(s)
      • Relação de usuário vinculado a campanha
    • Total / Restantes
      • Total de clientes incluido na campanha
      • Total de clientes restantes na campanha
      • Ao clicar nos valores, sistema redireciona para acessar os clientes cadastrados na campanha
    • Início Aleatório
      • Indicar a distribuição dos clientes para os usuários
    • Status
    • Botão: 
      • Permite o usuário editar as informações e parametros ca campanha
