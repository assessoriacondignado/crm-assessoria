TEMA: CONSULTAR CPF NO BANCO DE DADOS

Regras
  • ​​Aceita cpf com ou sem pontuação
  • Ao digitar Enter, sistema vai pesquisar
  • Caso não localize sistem avai cadastrar para INSS ou Siape ou outro orgão pré- cadastrado
  • Caso não localize o cadastro, deve verificar com o suporte da Assessoria se seu usuáiro está
  • habilitado para consulta de cadastro aleatório
===================================================================​

Fluxo
1º -> Acesse o menu Banco de Dados

2º -> Coloque o cpf na lupa

2º ->CRM vai tentar cadastrar o cliente sua base do INSS, SIAPE ou banco de dados interno

3ª -> CRM vai localizar e apresentar o cliente

​4º -> Caso não localize em noss banco de dados, será necessário verificar junto ao suporte se seu CPF está habilitado para atualizalização cadastral aleatória
     -> será necessário o uso da Integração ATUALIZAÇÃO CADASTRAL
